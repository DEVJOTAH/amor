Coloque aqui os arquivos de música (.mp3 é o mais compatível).

Depois, liste cada música em js/config.js, no array "playlist":

  playlist: [
    { arquivo: "nossa-musica.mp3", titulo: "Nossa Música", artista: "Artista" },
    { arquivo: "outra.mp3", titulo: "Outra Faixa", artista: "Outro Artista" },
  ]

O nome em "arquivo" precisa ser IGUAL ao nome do arquivo salvo
nesta pasta (incluindo maiúsculas/minúsculas).
