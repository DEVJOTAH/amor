Coloque aqui os arquivos de vídeo (.mp4 é o mais compatível com navegadores).

Depois, liste cada vídeo em js/config.js, no array "videos":

  videos: [
    { arquivo: "reveillon.mp4", titulo: "Réveillon 2024", data: "01/01/2024" },
  ]

O nome em "arquivo" precisa ser IGUAL ao nome do arquivo salvo
nesta pasta (incluindo maiúsculas/minúsculas).
